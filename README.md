## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/account-menu-pop-up.git
```

Go to the project directory

```bash
  cd account-menu-pop-up
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
